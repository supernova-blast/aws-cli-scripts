# AWS CLI Scripts

This repository provides a collection of Bash and Python scripts using AWS CLI to simplify everyday AWS operations.

## Features

- Scripts for common AWS resource management tasks
- Written in Bash and Python for flexibility and ease of use
- Useful for automation, reporting, and troubleshooting

## Usage

1. Browse the Bash (.sh) and Python (.py) scripts.
2. Ensure you have the AWS CLI configured and required permissions.
3. Run the scripts and customize as needed for your environment.
